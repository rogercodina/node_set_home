Node set home
=============

Allow users with module's permission to force a node from module's predefined list
to be homepage. Usage instructions:

1) Enable module

2) Go to "Administration » Configuration » System » Node Set Home" and then enter a list
   of nodes which will be selectable for homepage. Just enter node id for each node.

   Shortcut to page: admin/content/define-homepage

3) Go to permissions page "Administration » People » Permissions" and then assign
   permission 'Mark a node as homepage' to desired roles.

   Shortcut to page: admin/people/permissions

4) As admin or as user with role 'Mark a node as homepage' go to "Administration » Content
   » Define homepage". Then select your desired homepage.
